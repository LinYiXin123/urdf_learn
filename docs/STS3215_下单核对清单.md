# STS3215 下单核对清单

适用对象：
- 第一次买 `STS3215 / ST3215`
- 准备做你的机械臂项目
- 希望下单前先确认不要买错

## 1. 你这次应该买什么型号

推荐：

- `STS3215 / ST3215 7.4V C001`

不建议这次买：

- 普通 PWM 舵机
- 兼容 PWM 的其他总线舵机
- 不明型号众灵替代件

原因：

- 你当前模型路线更接近 `STS3215`
- 后面要做 leader/follower 与 LeRobot
- 这条路线最稳

## 2. 数量怎么下单

最稳顺序：

### 第一批

- `2 个`

用途：
- 打 `Y2.stl` 试样件
- 先验证舵机仓尺寸
- 先验证孔位和舵盘干涉

### 第二批

如果试装通过，再补：

- 单臂：再买 `4 个`
- 双臂：再买 `10 个`

## 3. 下单前必须确认的参数

卖家页面里必须确认：

- 是 `STS3215 / ST3215`
- 电压是 `7.4V`
- 是 `C001`
- 是总线舵机
- 是双轴输出

如果这些信息写不清楚，不建议下单。

## 4. 最重要的附件清单

每个舵机都必须确认是否带：

- `3-pin 舵机线`
- `舵盘`
- `舵盘螺丝`

如果页面没写清楚，直接问客服。

## 5. 你要直接问客服的话术

直接复制这段问：

```text
你好，我做机械臂项目，想确认下这个 STS3215 / ST3215 7.4V C001 舵机：
1. 是否是总线舵机？
2. 是否带 3-pin 连接线？
3. 是否带舵盘？
4. 是否带舵盘固定螺丝？
5. 是否是双轴输出？
6. 是否支持位置回读？
```

## 6. 到货后第一时间检查

收到货先不要装，先检查：

- 数量对不对
- 外壳有没有裂
- 舵机输出轴是否完好
- 线是否齐全
- 舵盘是否齐全
- 螺丝是否齐全

有缺件，先补件，别急着进装配。

## 7. 你当前最稳的购买策略

对你来说，最稳策略是：

1. 先买 `2 个`
2. 先做试装
3. 试装通过再补齐整套

这样最省返工成本。
