# 自有模型最低成本双臂具身执行说明

这份说明对应的最终执行包表格是：

- [自有模型_最低成本双臂具身_执行包.xlsx](F:/b站_Projects/urdf_learn/output/spreadsheet/自有模型_最低成本双臂具身_执行包.xlsx)

这份说明只服务一个目标：

`用你自己的 STL、二手 P2S、最低总成本路线，做出双臂具身采集原型。`

## 1. 已锁定路线

这版执行包已经把关键选择锁定了：

- 打印机：`二手 P2S`
- 打印材料：`PLA+ 1.75mm`
- 机械臂：`两套你自己的 STL`
- 舵机：`STS3215 / ST3215 7.4V C001`
- 控制软件：`LeRobot 主线`
- 目标：`双臂 + 双相机 + 采集/回放`

这版不追求：

- 第一版就做 AI 自主抓取
- 第一版就做腕部相机
- 第一版就做 Jetson 边缘部署

## 2. 预算口径

按执行包表格的默认预算：

- 双臂硬件与耗材：`约 1679.76 RMB`
- 二手 `P2S`：按 `3000–4000 RMB`
- 项目总预算：`约 4679.76–5679.76 RMB`

如果二手 `P2S` 超过 `4000 RMB`：

- 这条路线就不再是最低总成本
- 默认回退到第三方打印或 `A1 mini` 路线

## 3. 到货后的真实执行顺序

### 阶段 A：先验打印机

第一天只做打印机验机，不打印机械臂。

按这个顺序：

1. 开机自检
2. 检查热床加热
3. 检查喷嘴升温
4. 进料、退料
5. 打一个 `20mm` 校准块
6. 确认首层是否均匀

如果这里不过，不进入机械臂阶段。

### 阶段 B：先试样件，再整套打印

不要一开始就打完整两套。

先打印：

- `Y2.stl`

原因：

- 件小
- 成本低
- 更适合作为第一批“舵机仓/孔位/螺丝/舵盘干涉”的试装件

先拿 `2 个` 舵机做试装，确认：

- 舵机能装入
- 螺丝能锁上
- 舵盘不干涉
- 线有出口

试样件通过后，再打完整第一套。

### 阶段 C：先装 follower

第一台臂固定作为 `follower`。

顺序固定：

1. 清理全部打印件支撑和毛边
2. 单独给 6 个舵机设 `1–6` 号 ID
3. 按底座到夹爪顺序装配
4. 控制板接电脑
5. 电源接控制板
6. 单关节测试

只要 follower 跑通，项目就已经过了一半。

### 阶段 D：复制第二台做 leader

第二台直接复制第一台的装法。

不同点只有：

- 逻辑角色是 `leader`
- 仍然设 `1–6` 号 ID
- 标定和 LeRobot 中的设备类型不同

你现在的模型更像 `follower` 结构，所以第二台作为 `leader 原型` 是可用妥协，不是最完美的人体工学方案。

### 阶段 E：双相机与采集

双相机先用桌面外置位置，不做腕部支架。

建议固定：

- `front`：正前方俯视工作区
- `side`：侧前方观察夹爪

固定好之后，不要频繁挪动，否则每次都要重新适配采集环境。

## 4. 你接下来具体要填什么

执行包表格里有 4 张你后面要实际填写的记录表：

- `打印参数记录`
- `试装验收`
- `舵机ID表`
- `双相机映射`

你后面每推进一步，就把表格填一步，不要等最后再补。

## 5. LeRobot 路线怎么用

执行包里已经给了固定流程：

1. 安装 `Miniforge`
2. 创建 `lerobot` 环境
3. 安装 `ffmpeg`
4. `pip install "lerobot[feetech]"`
5. 用 `lerobot-find-port` 找两个串口
6. 分别 setup 两台臂
7. 分别 calibrate
8. 录 `5` 个 episode
9. 回放 `episode 0`

你第一版的成功标准不是“抓起来”，而是：

- 双臂能识别
- 双相机能采图
- 数据能采
- 数据能回放

## 6. 第一版一定不要做的事

为了控制时间和成本，第一版禁止做下面这些：

- 不改舵机型号
- 不换成兼容 PWM 的其他品牌
- 不加腕部相机
- 不加 Jetson
- 不一开始就训练抓取策略
- 不一开始就重新设计 leader 外壳

这些都会显著拉长项目周期。

## 7. 第一版验收标准

你最终要达到的最低交付标准是：

- follower 的 6 个舵机都能识别
- leader 的 6 个舵机都能识别
- 两个相机都能稳定采图
- LeRobot 可以录制至少一次 `5–10 分钟` 采集
- 采集数据可以正常回放

到这里，你的“最低成本双臂具身采集原型”就算完成了。
