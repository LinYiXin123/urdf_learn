## 1. 先说结论

你现在最稳的路线不是一次做完整双臂，而是：

1. 先把你自己的模型打印成一套
2. 先装 `1 台单臂`
3. 跑通接线、供电、控制、开合夹爪
4. 确认没问题后，再打印第 `2 套`，复制成第二台臂
5. 第二台后续作为 `leader` 原型使用

这样最省钱，也最不容易一次踩坑。

## 2. 你的模型大概有多大

按本地对 STL 的计算，这套模型整体约为：

- `8.263 x 11.928 x 51.409 cm`

这是标准桌面半米级机械臂的体量，和 SO-ARM100 量级一致，不是缩放错单位。

## 3. 推荐硬件

### 3.1 单臂硬件

建议单臂先买：

- `STS3215 / ST3215 7.4V C001` 总线舵机 `6 个`
- 总线舵机控制板 `1 块`
- `5V` 电源适配器 `1 个`
- `USB-C` 数据线 `1 根`
- 桌夹 `2 个`
- 小号十字螺丝刀套装 `1 套`
- `UVC` 摄像头 `1 个`
- 备用紧固件包 `1 套`

### 3.2 双臂具身采集硬件

完整具身采集再补：

- 上面整套硬件再来一份
- 第二个 `UVC` 摄像头
- 现有电脑一台

### 3.3 为什么推荐这类舵机

推荐 `STS3215 / ST3215`，原因：

- 尺寸和你当前模型路线最接近
- 是总线舵机，能回读位置
- 有磁编码器，适合后续做 leader/follower
- 国内可买，官方 CN BOM 可验证

不建议：

- 普通 PWM 舵机
- 你前面发过的众灵兼容 PWM 总线舵机

因为对你现在这套模型来说，机械尺寸和协议都不够稳。

## 4. 打印怎么交给第三方

你把 STL 发给打印商时，直接附这段要求：

```text
打印工艺：FDM
材料：PLA+
喷嘴：0.4mm
层高：0.2mm
填充：15%
要求：不要让支撑堵住横向螺丝孔；舵机安装位和孔位优先保证尺寸
先打一个试样件，不要直接全量开打
```

### 4.1 为什么要先打试样

因为你最容易遇到的问题不是“形状不对”，而是：

- 舵机装不进去
- 孔位堵住
- 螺丝穿不过
- 舵盘位置卡住

所以建议先让打印商打一个和舵机安装相关的试样件，再决定是否全量打印。

## 5. 你这 7 个 STL 的建议理解

这套命名不是官方名字，所以我给你的是“装配上的建议理解”，不是 100% CAD 官方命名。

按位置和形态看，大致可以理解为：

- `Y1.stl`：底座主件
- `YG.stl`：底座附近的小附件件
- `Y2.stl`：肩部/第二层连接件
- `Y4G.stl`：上臂段
- `Y5.stl`：前臂或腕前连接段
- `Y6G.stl`：腕部主件或固定夹爪侧
- `Y7G.stl`：活动夹爪侧

这份理解足够指导你“从底座一路装到夹爪”，但如果你后面要做到非常严谨的 CAD 对应，建议再做一次单件映射核对。

## 6. 装配前准备

在真正开始装之前，先完成这几件事。

### 6.1 准备工具

- 十字螺丝刀
- 小钳子
- 美工刀或去毛刺刀
- 扎带或魔术贴
- 记号贴纸

### 6.2 检查舵机附件

每个舵机都要检查是否带：

- 3-pin 线
- 舵盘
- 舵盘螺丝

如果卖家发的是裸机，先别装，先补齐附件。

### 6.3 准备备用紧固件

建议额外备：

- `M3x6` 盘头螺丝
- `M2.5x4` 小盘头螺丝
- `M3` 六角螺母

这些不是一定都会用完，但一旦少一颗，装配会很难受。

## 7. 装之前先设舵机 ID

不要一上来就把 6 个舵机全部装进去。

正确顺序是：

1. 先只接 `1 个` 舵机到控制板
2. 给它设一个唯一 ID
3. 再换下一个舵机
4. 最后让 6 个舵机的 ID 变成 `1, 2, 3, 4, 5, 6`

原因：

- 多个舵机同 ID 串在一起时，后面很难排错
- 新手最容易在这一步出错

### 7.1 最简单的做法

每次只接一个舵机：

- 控制板 -> 舵机

设完 ID 后，贴个小标签：

- `1`
- `2`
- `3`
- `4`
- `5`
- `6`

## 8. 装配总原则

每装一个关节，都按同样顺序：

1. 试装打印件
2. 让舵机回中位
3. 装舵机本体
4. 装舵盘
5. 接线
6. 锁螺丝

不要跳步。

特别是：

- 不要舵机没回中位就装舵盘
- 不要忘了先穿线再锁死
- 不要一次把所有螺丝锁满再发现线没走对

## 9. 单臂装配顺序

### 第 1 步：清理打印件

把所有打印件先清理干净：

- 去支撑
- 去毛边
- 通孔

如果孔里有残料，不要硬拧螺丝。

### 第 2 步：先装底座

先用 `Y1.stl` 作为底座主体。

如果 `YG.stl` 是底座附件，就先不要急着上，放到后面确认结构时再装。

先完成：

- 底座放平
- 第一个舵机装入底座位
- 固定牢

### 第 3 步：装肩部连接件

把 `Y2.stl` 作为第二层连接件去装。

这一步的重点不是快，而是：

- 先看孔位是否对正
- 确认舵盘不会干涉
- 线材有出口

### 第 4 步：装上臂

把 `Y4G.stl` 装到肩部上方。

装完后，轻轻手动检查：

- 是否有明显卡死
- 是否有结构干涉

### 第 5 步：装前臂

把 `Y5.stl` 接到上臂末端。

这是机械臂长度开始明显拉开的地方，注意：

- 线不要被夹到
- 舵机不要歪装

### 第 6 步：装腕部主件

把 `Y6G.stl` 装上去。

这一段通常最容易和夹爪产生干涉，所以先别一次锁死，先试活动范围。

### 第 7 步：装活动夹爪

把 `Y7G.stl` 作为活动夹爪去装。

装好后，检查：

- 开合是否顺畅
- 两边是否碰撞

### 第 8 步：补装小附件

如果 `YG.stl` 是底座小件或辅助件，这一步再补上。

## 10. 接线怎么接

最简单的总线接法：

```text
控制板 -> 舵机1 -> 舵机2 -> 舵机3 -> 舵机4 -> 舵机5 -> 舵机6
```

然后：

- 控制板 USB 接电脑
- 控制板电源接适配器

### 10.1 接线原则

- 先短接，后整理
- 线不要绷得太紧
- 每个关节活动时，线都要留余量

### 10.2 新手最常见错误

- 插反线
- 多个舵机 ID 一样
- 没有先单独测试每个舵机
- 线被关节夹住

## 11. 第一次上电怎么做

第一次上电只做最小测试。

### 11.1 上电前检查

- 每个舵机 ID 是否唯一
- 每个舵机线是否插牢
- 舵盘是否锁紧
- 打印件有无裂纹
- 底座是否夹稳

### 11.2 上电顺序

1. 控制板先接电脑
2. 再接电源
3. 识别舵机
4. 一次只测试一个关节

### 11.3 第一次测试目标

第一次不要追求整机动作，只要做到：

- 6 个舵机都能被识别
- 每个关节都能单独转
- 夹爪能开合

这就说明单臂已经成功 80% 了。

## 12. 怎么扩成完整具身系统

标准具身采集最少需要：

- `1 台 follower`
- `1 台 leader`
- `2 个 UVC 摄像头`
- `1 台电脑`

### 12.1 我建议的升级顺序

1. 先把单臂 follower 跑通
2. 再打印第 2 套模型
3. 再组第 2 台臂
4. 再加双相机
5. 再开始做采集和标定

### 12.2 最诚实的提醒

你现在这套模型更像标准工作臂结构，不一定是“专门适合人手示教”的 leader 结构。

也就是说：

- 第二套可以先当 `leader 原型`
- 但它未必有官方 leader 打印件那种更舒服的手感

这不影响你入门，但要提前知道。

## 13. 你现在最该做什么

如果你完全没入门，按这个顺序就够了：

1. 先下单 `2 个 STS3215`
2. 先让打印商打一个试样件
3. 试样件能装进去，再打整套
4. 再把 `6 个舵机` 补齐
5. 先装 `1 台单臂`
6. 跑通后再决定第 `2 台`
