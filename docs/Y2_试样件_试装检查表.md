# Y2 试样件试装检查表

适用场景：
- 你已经拿到打印机或打印商样件
- 你已经有 `2 个 STS3215`
- 现在准备做第一次试装

目标：
- 用最小成本确认你的模型是否适合这套舵机

## 1. 为什么先试 Y2

因为它更适合拿来验证：

- 舵机仓尺寸
- 孔位尺寸
- 舵盘干涉
- 线材出口

在完整打印前做这一步，可以大幅降低返工概率。

## 2. 试装前准备

你需要：

- `Y2.stl` 打印件
- `2 个 STS3215`
- 舵盘
- 舵盘螺丝
- 十字螺丝刀
- 卡尺

## 3. 试装顺序

### 第一步：看打印件

先看：

- 有没有裂纹
- 有没有明显翘曲
- 孔位是否被堵
- 舵机仓有没有残留支撑

### 第二步：空装舵机

不装螺丝，先把舵机试着放进去。

结果判断：

- `理想`：能顺利装入，略紧但不需要暴力
- `偏紧`：能进去，但明显吃力
- `不合格`：根本进不去，或者必须硬压

### 第三步：试孔位

把螺丝对准孔位试一下：

- 能不能顺利进入
- 是否明显偏孔

### 第四步：试舵盘

装上舵盘后，看看：

- 舵盘会不会碰到打印件
- 是否有明显歪斜
- 活动时是否刮蹭

### 第五步：试线材出口

把 3-pin 线插上，检查：

- 线能不能自然走出来
- 是否容易夹线
- 转动时是否会绷紧

## 4. 通过标准

只要满足下面 4 条，就算试样通过：

- 舵机能装入
- 螺丝能锁
- 舵盘不明显干涉
- 线材有出口且不容易夹住

## 5. 不通过时怎么处理

### 情况 1：舵机太紧

先处理：

- 去毛边
- 清支撑
- 轻微修整舵机仓

如果还是不行：
- 不要全量打印
- 先改打印参数或重新打试样

### 情况 2：螺丝孔堵住

处理：

- 清孔
- 去毛边
- 再试

### 情况 3：舵盘干涉

处理：

- 检查是否装歪
- 检查舵盘型号是否对
- 确认打印件是否局部变形

### 情况 4：线走不出来

处理：

- 重新调整试装方向
- 看是否需要线从另一侧绕出
- 确认这件是否真的是该关节件

## 6. 试样通过后才能做什么

只有 `Y2` 试样通过后，才建议继续：

1. 打完整第一套
2. 补齐剩余舵机
3. 开始单臂正式装配

如果试样没通过，不要直接进入整套打印。
