适用对象：
- 完全新手
- 第一次做机械臂
- 使用你自己的 STL 模型
- 使用 `STS3215 / ST3215 7.4V C001`
- 目标是先装出 `1 台能动的机械臂`，再复制出第 `2 台` 做双臂具身采集

适用模型文件：
- `归档/Y1.stl`
- `归档/Y2.stl`

- `归档/Y5.stl`
- `归档/Y6G.stl`
- `归档/Y7G.stl`
- `归档/YG.stl`

## 1. 先说最重要的结论

这套方案第一版：
- `不需要焊接`
- `不需要自己画板`
- `不需要自己做电机驱动`

第一版只做这些事：
- 打印零件
- 处理打印件
- 给舵机设 ID
- 把舵机装进打印件
- 把舵机线串起来
- 把控制板接电脑和电源
- 测试每个关节

## 2. 你现在需要准备什么

### 2.1 必须硬件

单臂必须：
- `STS3215 / ST3215 7.4V C001` 总线舵机 `6 个`
- 总线控制板 `1 块`
- 配套电源 `1 个`
- USB 数据线 `1 根`
- 桌夹 `2 个`
- UVC 摄像头 `1 个`

双臂完整具身采集：
- 再补一套上面的单臂硬件
- 第二个 UVC 摄像头
- Windows 电脑 `1 台`

### 2.2 必须工具

建议一次买齐：
- 十字螺丝刀套装
- 小钳子或斜口钳
- 美工刀或去毛刺刀
- 卡尺
- 扎带或魔术贴理线带
- 记号笔
- 小标签贴纸

### 2.3 建议准备的备用件

建议额外买一包：
- `M3x6` 盘头螺丝
- `M2.5x4` 小盘头螺丝
- `M3` 六角螺母

原因：
- 卖家不一定给全附件
- 装配中很容易掉螺丝
- 打印件孔位稍紧时，备用件能帮你排错

## 3. 你的零件大概对应什么位置

因为你的 STL 文件名不是官方名称，所以这里给的是“装配建议映射”。

建议按下面理解：
- `Y1.stl`：底座主件
- `YG.stl`：底座小附件件
- `Y2.stl`：肩部连接件
- `Y4G.stl`：上臂件
- `Y5.stl`：前臂或腕前连接件
- `Y6G.stl`：腕部主件或固定夹爪一侧
- `Y7G.stl`：活动夹爪一侧

如果实物对不上，优先按这个原则判断：
- 离地最近、最宽的件是底座
- 中间最长的件一般是上臂和前臂
- 最末端最小的一组一般是腕部和夹爪

## 4. 打印要求

推荐参数：

```text
材料：PLA+
喷嘴：0.4mm
层高：0.2mm
填充：15%
```

### 4.1 打印顺序

不要直接打两整套。

先这样打：
1. `Y2.stl` 试样件
2. 用 `2 个` 舵机试装
3. 试装通过后再打完整第一套
4. 第一套装好、测试通过后，再打第二套

### 4.2 为什么先打 Y2

因为它通常更适合作为：
- 舵机仓验证件
- 螺丝孔验证件
- 舵盘干涉验证件

这一步通过了，后面整套出问题的概率会明显下降。

## 5. 打印完成后的后处理

每个打印件都要先做这几件事：

1. 去支撑  
2. 去毛边  
3. 清理孔位  
4. 清理舵机安装位  
5. 检查有没有裂纹  

### 5.1 正确处理方式

- 小毛边：用美工刀修
- 孔位堵塞：轻轻刮开，不要硬拧螺丝
- 舵机仓太紧：先试插，不要蛮力压
- 有裂纹：不要继续装，直接补打

## 6. 舵机到货后先不要装

第一步不是装配，而是：

`先给舵机编号和设 ID`

### 6.1 为什么先设 ID

如果 6 个舵机都默认一个 ID，串起来以后会很难排错。

### 6.2 正确做法

每次只接一个舵机：

```text
控制板 -> 单个舵机
```

然后依次设：
- 舵机 1 -> ID 1
- 舵机 2 -> ID 2
- 舵机 3 -> ID 3
- 舵机 4 -> ID 4
- 舵机 5 -> ID 5
- 舵机 6 -> ID 6

设完后用标签贴纸贴在舵机上。

第二台机械臂也按同样方式做。

### 6.3 follower / leader 的 ID 规则

建议统一规则：

follower：
- shoulder_pan -> 1
- shoulder_lift -> 2
- elbow_flex -> 3
- wrist_flex -> 4
- wrist_roll -> 5
- gripper -> 6

leader：
- shoulder_pan -> 1
- shoulder_lift -> 2
- elbow_flex -> 3
- wrist_flex -> 4
- wrist_roll -> 5
- gripper -> 6

注意：
- 两台臂不要同时接在一块控制板上设 ID
- 每次只连接一台臂

## 7. 装配的总原则

每装一个关节都按这个固定顺序：

1. 试装打印件  
2. 舵机回中位  
3. 装舵机本体  
4. 装舵盘  
5. 接线  
6. 锁螺丝  

不要跳步。

特别注意：
- 舵机没回中位前不要装舵盘
- 线没走好前不要锁死所有螺丝
- 不要把线拉得很紧
- 不要让线走到关节活动范围里

## 8. 单臂详细装配顺序

下面以第一台 `follower` 为例。

### 第 1 步：摆好全部零件

把所有 STL 打印件摆在桌上，建议按从大到小排：
- `Y1`
- `YG`
- `Y2`
- `Y4G`
- `Y5`
- `Y6G`
- `Y7G`

同时把 6 个舵机、舵盘、螺丝、控制板、电源、USB 线都摆好。

### 第 2 步：装底座

先拿 `Y1.stl`。

操作：
1. 确认底座能平放
2. 找到底座上的舵机安装位
3. 用 `ID 1` 舵机作为底座关节舵机
4. 先试装，不要先锁满螺丝
5. 确认线有出口
6. 再锁螺丝

如果 `YG.stl` 是底座小附件，这一步先不装，最后补装。

### 第 3 步：装肩部件

拿 `Y2.stl`。

操作：
1. 把 `ID 2` 舵机装进对应安装位
2. 先试装 Y2 与底座之间的连接
3. 检查孔位和舵盘是否干涉
4. 确认线可以从底座引到肩部
5. 再锁紧

### 第 4 步：装上臂

拿 `Y4G.stl`。

操作：
1. 用 `ID 3` 舵机
2. 先装舵机，再装 Y4G
3. 轻轻摆动，检查是否有卡住

### 第 5 步：装前臂

拿 `Y5.stl`。

操作：
1. 用 `ID 4` 舵机
2. 接到上臂末端
3. 检查这一段线是否容易被挤压

### 第 6 步：装腕部

拿 `Y6G.stl`。

操作：
1. 用 `ID 5` 舵机
2. 先试活动范围
3. 确保腕部转动时线不会夹住

### 第 7 步：装夹爪

拿 `Y7G.stl`。

操作：
1. 用 `ID 6` 舵机
2. 先检查夹爪开合方向
3. 安装后轻推活动件，确认不碰撞

### 第 8 步：补装小附件

如果 `YG.stl` 是底座小附件或补强件，这一步再装。

## 9. 舵盘怎么装

这是新手最容易装错的地方。

### 9.1 装舵盘前必须做的事

先让舵机处于中位。

简单理解就是：
- 先通电
- 先让舵机回一个标准中间位置
- 再把舵盘拧上去

### 9.2 为什么要这样

如果舵盘装歪：
- 后面零位会偏
- 有的关节会提前顶死
- 夹爪开合会异常

### 9.3 装好后的检查

装完一个舵盘，就做一次检查：
- 舵盘是否明显歪斜
- 是否碰到打印件
- 是否锁紧

## 10. 总线接线怎么接

最简单的接法是串接。

### 10.1 单臂接线

```text
控制板 -> 舵机1 -> 舵机2 -> 舵机3 -> 舵机4 -> 舵机5 -> 舵机6
```

再接：

```text
电脑 -> USB -> 控制板
电源 -> 控制板
```

### 10.2 双臂接线

两台臂分开：

```text
电脑 -> 控制板A -> follower 6个舵机
电脑 -> 控制板B -> leader 6个舵机
```

相机也是独立接电脑：

```text
电脑 -> front 相机
电脑 -> side 相机
```

### 10.3 布线原则

- 每个关节转动时线都有余量
- 线不要贴着齿轮或活动边缘
- 线不要被夹在两片结构之间
- 用扎带或魔术贴整理，但不要勒太紧

## 11. 第一次上电步骤

第一次上电只做最小测试。

### 11.1 上电前检查

逐项检查：
- 舵机 ID 是否唯一
- 线是否插牢
- 舵盘是否锁紧
- 打印件是否有裂纹
- 底座是否夹稳
- 线有没有被夹住

### 11.2 上电顺序

1. 控制板先接电脑
2. 再接电源
3. 识别舵机
4. 一次只动一个关节

### 11.3 第一次测试目标

第一次不要追求整机联动。

只要做到：
- 6 个舵机都被识别
- 每个关节单独能动
- 夹爪能开合

这就算通过。

## 12. 新手排错表

### 问题 1：舵机不识别

优先检查：
- 电源有没有接上
- USB 是否是数据线
- 舵机 ID 是否冲突
- 线有没有插反

### 问题 2：舵机识别了但不动

优先检查：
- 是否真的发了控制命令
- 舵机是不是被机械卡死
- 舵盘装得太歪

### 问题 3：关节一动就卡

优先检查：
- 舵盘干涉
- 打印件毛边没处理干净
- 螺丝锁得过紧
- 线被夹住

### 问题 4：夹爪打不开

优先检查：
- `Y6G` 和 `Y7G` 是否装反
- 舵盘零位是否偏
- 两边是否相互碰撞

## 13. 第二台机械臂怎么做

第一台 follower 跑通后，再做第二台。

第二台装配方式与第一台几乎相同，只是角色变成 `leader`。

建议：
- 完全复制第一台的打印参数
- 完全复制第一台的装配顺序
- 先保证“能用”，再考虑“更适合手操控”

## 14. 相机怎么装

第一版不要折腾腕部相机。

先用外置双相机：

### front 相机
- 放在机械臂前上方
- 能看到工作区全貌

### side 相机
- 放在侧前方
- 能看到夹爪接近物体

先固定住，不要随便挪。

## 15. 双臂具身采集怎么接

要达到第一版“完整具身采集原型”，只需要做到：

- follower 能动
- leader 能动
- front 相机能采图
- side 相机能采图
- LeRobot 能录和放

你不需要第一版就实现：
- AI 自主抓取
- 复杂策略训练
- 端侧 Jetson 推理

## 16. 你现在最该怎么开始

最稳顺序：

1. 买 `P2S`
2. 买 `2 个 STS3215`
3. 打 `Y2.stl` 试样件
4. 试装通过后买齐剩余舵机和配件
5. 打完整第一套
6. 装 follower
7. 跑通后再打第二套
8. 装 leader
9. 接双相机
10. 跑 LeRobot 采集链路
